Part 3: Bootstrap and HTML

In this portion of the assignment, you will be creating an interactive form for your website.

The goal of this form is to allow you as the site admin to upload new files and articles to the website without needing to go into the backend.

Step 0 :
Create an HTML file and name it form.html. This is where you will build the form page.
Step 1:
Once you have set up the basics in your html file and included Bootstrap, start by making a new <form> object. Bootstrap has a suite of options for use with forms.
Step 2:
Add two text input form-groups, one for the title and the other for the author of the article being written.
Step 3:
Next, you’ll use Bootstrap’s form-row class to align a file upload widget and a select or a dropdown for what page the article will appear on.
Step 4:
Add an area for the user to input the article text. This should be a large box rather than a thinner one like we used for the title.
Step 5:
Horizontally align a set of inline checkboxes with a radio button to allow the user to add meta tags to their article and determine whether it will be publicly visible.
Step 6:
Create a login portal to verify the identity of the person trying to upload to the site. This is a combination of a text input, a password input, a checkbox, and a button.
